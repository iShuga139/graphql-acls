<a name="Development Installation"></a>
## Development Installation

1. Clone the repository:

```bash
  $ git clone https://github.com/iShuga139/graphql-acls.git
  $ cd graphql-acls
```

2. Install dependencies:

  ```bash
  $ npm install
  ```

3. Run the application:

  ```bash
  $ npm start
  ```
